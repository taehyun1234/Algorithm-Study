코루틴
- 메인 루틴 안에서 특정 시간 후에 실행되도록 하는 서브루틴을 말한다.
- 특정 시간 후에 코루틴을 실행하고 나서, 코루틴이 끝나면 메인루틴이 다시 진행되도록 한다.

코루틴과 쓰레드의 차이점
즉, 코루틴은 Single Core로 Multi Process 방식으로 사용하고, 쓰레드는 Multi core로 Multi Thread 방식으로 사용한다고 생각하면 될 것 같다.
그러므로, 코루틴의 경우 동기화를 해 줄 필요가 없고 반면, 쓰레드는 객체 간의 자원을 공유함으로써 발생하는 문제를 해결하기 위해 동기화를 해줘야 한다.
코루틴을 경량 쓰레드라고 생각해도 좋을 것 같다. 

라고 생각했는데, 좀 더 찾아봐야할 것 같다. 유니티에서의 코루틴 내용은 아직 좀 더 공부해봐야할듯.
